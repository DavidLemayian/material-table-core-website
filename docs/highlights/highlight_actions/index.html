Actions

You can add buttons to rows or toolbar by using actions prop. Actions prop takes array of actions.

Usage#

<MaterialTable
  // other props
  actions={[
    {
      icon: 'save',
      tooltip: 'Save User',
      onClick: (event, rowData) => {
        // Do save operation
      }
    }
  ]}
/>

Demo#

Live Editor
function ActionsDemo() {
  return (
    <MaterialTable
      data={[{ id: 2, name: "Joe" }, { id: 1, name: "Mary" }]}
      columns={[{ field: "id", title: "Id" }, { field: "name", title: "Name" }]}
      actions={[
        {
          icon: 'save',
          tooltip: 'Save User',
          onClick: (event, rowData) => {
            console.log({ event, rowData });
          }
        }
      ]}
    />
  );
}
