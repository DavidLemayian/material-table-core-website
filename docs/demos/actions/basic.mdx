You can add buttons to rows or toolbar by using actions prop. Actions prop takes array of actions.

## Usage

```jsx
<MaterialTable
  // other props
  actions={[
    {
      icon: 'save',
      tooltip: 'Save User',
      onClick: (event, rowData) => {
        alert("Do save operation : " + JSON.stringify(rowData, null, 2));
      }
    }
  ]}
/>
```
