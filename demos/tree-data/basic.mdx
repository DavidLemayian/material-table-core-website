You can add buttons to rows or toolbar by using actions prop. Actions prop takes array of actions.

## Usage

```jsx
import MaterialTable from "@material-table/core";

<MaterialTable
  // ...
  parentChildData={(row, rows) => rows.find((a) => a.id === row.parentId)}
/>;
```
