# Basic Sorting

## Props

| Field                    | Type    | default | Description                                                             |
| ------------------------ | ------- | ------- | ----------------------------------------------------------------------- |
| `options.sorting`        | boolean | false   | Activate sorting functionality, display arrow icon when hovering header |
| `options.thirdSortClick` | boolean | true    | Allow unsorted state on third header click                              |
| `column.sorting`         | boolean | true    | Set as `false` to remove sorting option in a column                     |

## Usage

```jsx
import MaterialTable from "@material-table/core";

<MaterialTable
  // ...
  options={{
    sorting: true,
    thirdSortClick: false,
  }}
  columns={[
    {
      title: "Name",
      field: "name",
      sorting: false,
    },
    {
      title: "Surname",
      field: "surname",
    },
    {
      title: "Birth Year",
      field: "birthYear",
    },
    {
      title: "Birth Place",
      field: "birthCity",
      lookup: {
        34: "İstanbul",
        63: "Şanlıurfa",
      },
    },
  ]}
/>;
```
